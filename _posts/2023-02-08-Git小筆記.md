- gitignore重新生效

```bat
git rm -r --cached . #清除快取
git add . #重新trace file
```

- git 遠端儲存庫

```bat
git remote -v #查看目前設定
git remote add <name> <url> #新增遠端儲存庫
git push <name> <branch name> #推送至遠端儲存庫
git pull #拉取遠端儲存庫資料並合併
```
