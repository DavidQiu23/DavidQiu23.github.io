## ASP.NET Core：自定義 Web API 驗證錯誤訊息

在 ASP.NET Core 中，使用 `[ApiController]` 屬性會自動對請求進行模型驗證（Model Validation）。若驗證失敗，系統預設會回傳 400 BadRequest 錯誤。

根據官方文件的 [自動 400 回應方法](https://learn.microsoft.com/zh-tw/aspnet/core/web-api/?view=aspnetcore-3.1)，我們可以自定義這些預設的回覆訊息，並同步記錄相關日誌。

**注意事項：**
由於 HTTP 的 `Request.Body` 屬於 `Stream` 類型且預設僅能讀取一次，若要在驗證失敗時讀取 Body 內容，需在中介層（Middleware）預先啟用緩存機制，並在讀取時開啟同步 IO 支持。

### 設定自定義驗證回應 (Startup.cs)

```csharp
services.AddControllers().ConfigureApiBehaviorOptions(options =>
{
    // 取得原先系統預設的處理委派 (Delegate)
    var builtInFactory = options.InvalidModelStateResponseFactory;

    options.InvalidModelStateResponseFactory = context =>
    {
        // 啟用同步 IO，否則 StreamReader.ReadToEnd() 可能會失敗
        var syncIoFeature = context.HttpContext.Features.Get<IHttpBodyControlFeature>();
        if (syncIoFeature != null) syncIoFeature.AllowSynchronousIO = true;

        var logger = context.HttpContext.RequestServices.GetRequiredService<ILogger<Startup>>();

        // 將 HTTP Request Body 的串流位置歸零以便讀取
        context.HttpContext.Request.Body.Position = 0;
        using var reader = new StreamReader(context.HttpContext.Request.Body);
        var body = reader.ReadToEnd();

        // 記錄錯誤資訊至 ModelState 與日誌
        context.ModelState.AddModelError("RawRequestBody", body);
        logger.LogDebug($"Validation failed for request body: {body}");

        // 調用原先的行為產生回應（或在此自定義回傳 IActionResult）
        return builtInFactory(context);
    };
});
```

### 啟用緩存機制的中介層 (Middleware.cs)

```csharp
public async Task Invoke(HttpContext httpContext)
{
    try
    {
        // 啟用緩存機制，允許 Request.Body 被多次讀取
        httpContext.Request.EnableBuffering();
        await _next(httpContext);
    }
    catch (Exception e)
    {
        _logger.LogError(e, "發生未預期錯誤");
        
        var response = new { message = e.Message };
        httpContext.Response.ContentType = "application/json";
        httpContext.Response.StatusCode = StatusCodes.Status500InternalServerError;
        await httpContext.Response.WriteAsync(JsonSerializer.Serialize(response));
    }
}
```
